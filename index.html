Inside the grid container rule, we define how grid areas are laid out on the grid with the property :

grid-template-areas: "item1 item2 item3".

It means we create one grid row with three grid columns.

If we want to span a grid item across multiple grid columns, we repeat the name of the grid item like so:

grid-template-areas: "item1 item2 item2".

Aligning Grid Items

We use thejustify-items to align grid items along the row axis.

The default value is stretch means they fill all available space, like the first grid we've created.

Item 2

Like Flexbox, if we want to align grid items to the left edge of the grid area along the row axis, we use .grid-container {justify-items: start;}

By the way, it's endd for the right edge of the grid area, always along the row axis.

More importantly, center used to align the grid items to the center of the column axis

If we want to align grid items along the column axis, we set align-items to stretch to have them occupy the entire grid area

What happens when we set the align-items property to stretch?
